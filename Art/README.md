# Card Survivor - Recursos de Arte

## Descarga los assets manualmente:

### Sprites y Personajes
- **Kenney RPG Assets**: https://kenney.nl/assets/RPG
- **OpenGameArt Dungeon**: https://opengameart.org/content/dungeon-tileset
- **itch.io Gothic Pack**: https://itch.io/game-assets/free/tag-gothic

### UI y Frames
- **Craftpix Free GUI**: https://craftpix.net/freebies/
- **Game-Icons.net**: https://game-icons.net

### Fondos
- **CraftPix Dungeon Backgrounds**: https://craftpix.net/freebies/category-2d-pixel-art/
- **OpenGameArt Backgrounds**: https://opengameart.org/content/dungeon-background

## Propuestas de arte para Card Survivor (estilo gótico):

### Personajes
- Zombi, Murciélago, Golem, Jefe (Señor Sombras)
- Guerrero, Mago, Pícaro, Paladín

### Cartas
- Frame gótico dorado/negro
- Card backs con símbolos esotéricos

### UI
- Menú principal gótico (castillo/iglesia)
- Botones con ornamentación
- Icons:espada, escudo, poción, moneda

## Uso
1. Descarga los packs
2. Extrae en la carpeta correspondiente
3. Añade a Xcode como recursos

## Créditos
- Kenney.nl (CC0)
- Craftpix.net (gratis con atribución)
- OpenGameArt.org (varios autores)
